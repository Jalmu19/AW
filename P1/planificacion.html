Planificación del Proyecto

El desarrollo del proyecto Bistro FDI se organizará en varias fases que coinciden con las entregas parciales de la asignatura. El trabajo se repartirá entre los cuatro miembros del grupo de forma equilibrada.

Se realizarán reuniones periódicas para coordinar el trabajo, comprobar el avance y resolver posibles incidencias.


Fases del proyecto

Fase 1:

Creación de la web descriptiva del proyecto: páginas index, detalles, bocetos, miembros, planificación y contacto.

Fase 2:

Implementación de las funcionalidades 0, 1 y 2: gestión de usuarios, productos y pedidos. Incluye diseño de base de datos, vistas principales y pruebas.

Fase 3:

Implementación de la funcionalidad 3 (preparación de pedidos) y mejora de las funcionalidades anteriores. Se añadirán hojas de estilo CSS para mejorar la apariencia.

Fase 4:

Finalización del proyecto: integración de todas las funcionalidades, depuración de errores y optimización del rendimiento.
