Práctica 1: Creación de una página web

Descripción

Este proyecto consiste en una página web con la que se podrá gestionar el restaurante Bistro FDI.
Así, se podrán realizar varias acciones dependiendo del rol asignado en la web:

  • Cliente: podrá realizar pedidos y ver el estado de estos.
    Para ello debererá registrarse en la aplicación web y, una vez identificado, podrá acceder a su perfil
    y a todas las compras realizadas.
  • Trabajadores: podrán organizar la preparación de los pedidos de los clientes rápidamente.
    Cada trabajador cuenta con un rol específico. Encontramos camareros, cocineros y el gerente.
    Cada uno tendrá una interfaz distinta que le permita realizar su trabajo eficientemente.
